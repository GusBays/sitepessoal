Sobre

Sou uma pessoa muito tranquila, focada em meus objetivos, procuro absorver todo conhecimento possível e ajudar naquilo que for do meu alcance. Atualmente estou cursando ciência da computação na Universidade do Vale do Rio dos Sinos.

Meu primeiro contato com programação foi em 2017 com o Projeto SAP TechSchool Social. Esse projeto foi um curso fornecido pela SAP Labs Latin America que ocorreu nos meus 3 anos de ensino médio na escola Polisinos onde tivemos aulas focadas em inglês, programação e robótica.

As aulas de programação começaram no primeiro ano com o básico de HTML e CSS, no segundo ano vimos um pouco de Python e, no terceiro, vimos um pouco mais a fundo de banco de dados utilizando MySQL.

Este site foi construído durante a Imersão CSS da Alura.
